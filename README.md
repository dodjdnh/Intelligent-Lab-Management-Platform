# 智慧实验室管理系统 (Smart Lab Management System)

本项目是一款为高校实验室量身定制的数字化管理平台。通过全栈式架构实现实验室预约、耗材精细化管理及人员权限控制，旨在解决传统实验室管理中信息滞后、预约冲突及物资流转不透明等痛点。

---

## 🌟 项目亮点

* **全流程闭环：** 实现了从“学生申请 -> 管理员审核 -> 自动扣减库存/占用时段”的完整业务流。
* **权限安全控制：** 基于 **Sa-Token** 实现轻量级登录鉴权，支持管理员与学生角色权限分离。
* **数据一致性：** 针对耗材入库设计了自动合并逻辑，有效避免冗余数据。
* **体验优化：** 前端采用 **Vue3 + Element-Plus**，支持首页数据看板，实时监控待办任务及库存预警。

## 🛠️ 技术栈

### 后端 (Backend)
* **框架：** Spring Boot 3.x
* **数据库：** MySQL 8.0
* **持久层：** MyBatis-Plus (简化 CRUD 操作)
* **权限管理：** Sa-Token (Token 鉴权、角色校验)
* **工具：** Lombok, Maven

### 前端 (Frontend)
* **框架：** Vue 3 (Composition API)
* **组件库：** Element-Plus
* **网络请求：** Axios
* **图标库：** @element-plus/icons-vue

## 🚀 快速开始

### 1. 环境准备
* JDK 17+
* Node.js 16+
* MySQL 8.0+

### 2. 数据库初始化
1.  创建一个名为 `lab_db` 的数据库。
2.  运行项目根目录下 `/sql/lab_management.sql` 文件，完成表结构及初始数据导入。

### 3. 后端启动
1.  使用 IDEA 打开 `management` 目录。
2.  修改 `src/main/resources/application.yml` 中的数据库用户名和密码。
3.  运行 `ManagementApplication.java`。

### 4. 前端启动
1.  终端进入 `lab-system` 目录。
2.  安装依赖：`npm install`
3.  启动开发服务器：`npm run dev`
4.  访问浏览器：`http://localhost:5173`
